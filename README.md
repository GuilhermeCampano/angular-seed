# Seed for angular 1.5 with webpack

### Install global dependencies
```
npm i webpack -g
npm i typescript -g
npm i typings -g
```
Afterwards run
```
npm i && typings i
```
Start project in dev modus
```
npm run dev
```
Optional: Check for linting errors
```
npm run lint
```

### Enable tslint in webstorm
See https://www.jetbrains.com/webstorm/help/tslint.html
